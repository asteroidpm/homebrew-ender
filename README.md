# Ender Homebrew Tap

Public Homebrew tap for Ender's downloadable tools.

```sh
brew install asteroidpm/ender/ender-payables-mcp
# auto-taps github.com/asteroidpm/homebrew-ender and installs ender-payables-mcp-server
```

## Formulae

- `ender-payables-mcp` — Ender's public MCP server for creating payables (invoices)
  via the Ender API. See the
  [customer guide](https://docs.ender.com/customer-guides/public-api/mcp-server).
